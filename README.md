# Brewline Coffee Co.

A specialty cafe landing page built with HTML and CSS. Editorial aesthetic, warm neutral palette, fully responsive.

## About the project

Brewline Coffee Co. is a fictional specialty cafe based in Bengaluru. This is my first complete HTML/CSS project — built to practice layout, typography, and responsive design while producing something that looks production-quality.

The design goal was editorial warmth: the kind of page a real independent cafe would actually want.

---

## Sections

- Fixed glassmorphism navbar
- Hero with stats
- Scrolling marquee strip
- About / brand values grid
- Menu cards with hover states
- Ambience / seating area image gallery
- Testimonials
- Visit & contact with embedded Google Maps
- Footer

---

## Built with

- HTML5
- CSS3 (Grid, Flexbox, custom properties, keyframe animation)
- Google Fonts — Cormorant Garamond + DM Sans
- No frameworks, no JavaScript

## What I learned

- CSS Grid for complex two-column layouts
- `position: fixed` navbar with `backdrop-filter` blur
- CSS keyframe marquee animation
- Hover overlay technique on images using `position: absolute`
- Mobile-first media queries at `768px`
- Git + GitHub Pages deployment
